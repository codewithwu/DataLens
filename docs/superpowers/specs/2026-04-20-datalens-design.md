# DataLens 消费数据可视化工具 设计文档

## 1. 概述

- **项目名称**: DataLens
- **类型**: 纯前端单页应用
- **核心功能**: 上传 CSV 账单文件，可视化展示每日各时段消费数据，支持单天/多天对比
- **目标用户**: 查看 API 消费账单数据的用户

## 2. 技术栈

- **HTML + CSS + JavaScript** — 单文件，可直接双击在浏览器打开
- **Chart.js** — 图表可视化（柱状图、折线图）
- **PapaParse** — CSV 文件解析

## 3. 数据结构

### 3.1 输入数据（CSV）
- 文件路径: `/home/cooper/githubProjects/DataLens/export_bill_1776519761.csv`
- 关键字段:
  - `总消费数` — 数值（输入消费数 + 输出消费数）
  - `消费时间` — 格式: `2026-04-17 22:00-23:00`（日期 + 小时时段）

### 3.2 时间解析规则
- 原始格式: `YYYY-MM-DD HH:00-HH:00`
- 日期: 取 `YYYY-MM-DD` 部分
- 时段: 取 `HH:00-HH:00` 部分（如 `22:00-23:00`）
- 时段顺序: 按小时升序排列（00:00-01:00, 01:00-02:00, ...）

### 3.3 数据聚合
- 按时段聚合：同一小时内所有记录的总消费数汇总
- 多天对比：每日所有时段总消费数之和 = 每日总消费数

## 4. 功能模块

### 4.1 文件上传
- 拖拽上传或点击选择文件
- 仅接受 `.csv` 文件
- 上传后解析并显示日期列表

### 4.2 日期选择
- 多选下拉框列出所有有数据的日期
- 按钮触发图表渲染
- 支持单选和多选

### 4.3 图表渲染
| 选择 | 图表类型 | X 轴 | Y 轴 |
|------|---------|------|------|
| 单天 | 柱状图 | 时段（如 09:00-10:00） | 该时段总消费数 |
| 多天 | 折线图 | 日期 | 每日所有时段总消费数之和 |

- Hover tooltip 显示详细数据
- 响应式布局
